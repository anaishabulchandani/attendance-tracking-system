Attendance Tracking System 

A simple and efficient Attendance Tracking System built using Python, Streamlit, and basic Data Structures (Queue, Stack, Dictionary, List).
The system allows teachers to manage students, mark attendance (directly or via queue), and generate useful reports.

Features:
1. Student Management:

Add Student (Roll No + Name)
Delete Student
View Student List
Students are saved in students.json for persistence

2.Attendance Management

Mark Present/Absent directly
Add attendance tasks to a Queue (FIFO)
Process queue tasks one-by-one
View attendance for any selected date
Delete attendance for a date

3.Reports & Visualization

Student-wise attendance report
Date-wise present count chart
Clean and simple UI using Streamlit tabs

Data Structures Used:

Dictionary

Queue (FIFO)

Stack (LIFO)

Lists

Technologies Used:

Python 3.x

Streamlit (UI)

Pandas (tables + charts)

JSON (data persistence)
Matplotlib (diagram generation)
Pathlib (file handling)
